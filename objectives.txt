===== fuctionalities =====
- authentication (register/login)
- creating a ticket (outsiders)
- changing ticket status (internal users only)
- adding notes
- logging changes

===== tables =====
tickets:

- id
- title
- description
- status [new, acknowledged, in-progress, completed, cancelled, rejected]
- created_at
- created_by
- user_id
- username

ticketHistory:
- id
- ticket_id (fk)
- created_at
- user_id
- username
- status

ticketNotes:
- id
- ticket_id (fk)
- note
- created_at
- modified_at
- user_id
- username

users:
- id
- username
- email
- password
- created_on
- updated_on
- deleted
- roles
